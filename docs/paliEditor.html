Manual and Tips

paliEditor is the pali special character center and tipitaka tool center for tipitaka students.
Manual
paliEditor has 2 methods--mouse click and meyboard shortcut, to insert pali special character.
1. Keyboard Shortcut
--Use Shortcut Key to insert pali special character. See below "shortcut List" for list of shortcut keys.
2. Mouse Click
--Click on "Toolbar" then click on a character button that you would like to insert. Button sorted as alike as Samabhidhna in Kaccayanaveyykaraa.
-Another Tips
--Press Ctrl+A to select all message in the text area, then press Ctrl+C to copy.
--You should bookmark this page for easy to access.
--The editor will auto-save your message every 5 minute.
--You can use all functions of your browser.
--Turn search tool, in reference tool, off to tweak paliEditor faster.
--Don't open more than one paliEditor, because the save option can save only one paliEditor--except a difference browser user, browser application, or OS. If you have more than 1 article to write, you can use Seperate button in Toolbar tab to seperate between content.

Shortcut List

paliEditor has 2 choice of shortcut keys for you.
1. Cross Web Browser Shortcut (recommend) ,always use with Ctrl, support to all modern browsers such as chrome, firefox, internet explorer, safari, opera, midori.
2. Pali Relative Shortcut always use with Ctrl don't support in every browser, such as IE, but they are relative with Samabhidhna of Kaccayanaveyykaraa that make you easy to remember keys.

Cross Web Browser Shortcut

    "<save>" = Ctrl + Q
